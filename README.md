## Installation
### Clone
```
$ git clone https://github.com/azmprllynsa/mooichat.git
$ cd mooichat
$ npm install
```
### Create Environment Variable
```
# Set API KEY
VUE_APP_API=YOUR_HOST_URL
VUE_APP_SECRET_KEY_TOKEN=YOUR_SECRET_KEY
```
### Start Development Server
```
$ npm run serve
```
